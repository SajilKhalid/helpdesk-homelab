# Help Desk Homelab

A hands-on IT support lab built to simulate real-world help desk and troubleshooting scenarios using Windows 11 virtual machines.

This project demonstrates practical experience in diagnosing and resolving common IT issues while documenting each case in a structured, ticket-based format.

---

## 📌 What This Project Shows

* User account management and authentication troubleshooting
* Basic networking and system diagnostics
* Structured problem-solving and root cause analysis
* Clear documentation of IT support workflows

---

## 🧑‍💻 Lab Structure

Each lab is organized as an individual **support ticket**, where every ticket functions as a **mini project**.

Each ticket represents a complete troubleshooting scenario:

* A real-world problem is simulated
* The issue is investigated step-by-step
* A resolution is implemented
* The entire process is documented with screenshots

---

### 📁 Repository Structure

```
helpdesk-homelab/
├── labs/
│   ├── ticket-001-login-issue.md
│   ├── ticket-002-...
│
├── screenshots/
│   ├── ticket-001/
│   ├── ticket-002/
```

* Each `.md` file in `labs/` contains the full ticket write-up
* Each corresponding folder in `screenshots/` stores supporting images
* Screenshots are linked directly inside each ticket for clarity

---

## 🎯 Purpose

This lab is designed to simulate real IT support workflows and build hands-on experience relevant to:

* Help Desk / IT Support roles
* SOC Analyst (entry-level)
* Technical support positions

---

## 🚀 Ongoing Work

Additional tickets will be added to expand skills in:

* Password resets and account lockouts
* Network troubleshooting (DNS, connectivity issues)
* Security event analysis (logs, failed logins)
* Endpoint configuration and system management
